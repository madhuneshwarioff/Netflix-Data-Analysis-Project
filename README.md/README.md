Netflix Content Analysis Dashboard
Project Overview

In this project, I explored Netflix’s content library to understand how movies and TV shows are distributed across different countries, years, and genres. The goal was to uncover patterns in Netflix’s catalog, such as how content production has grown over time and which countries contribute the most titles.

The analysis was performed using Python for data cleaning and exploration, and the final insights were visualized through an interactive dashboard built in Microsoft Power BI.
Tools Used

Python

Pandas

Matplotlib

Microsoft Power BI

Key Analysis Performed

Cleaned and prepared the dataset for analysis

Performed exploratory data analysis (EDA) to understand the structure of the data

Analyzed genre distribution across Netflix titles

Studied how Netflix content has grown over the years

Examined global trends in content production by country

Dashboard Features

The interactive dashboard provides several visual insights, including:

Distribution of Movies vs TV Shows

Content growth over the years

Top countries producing Netflix content

Content rating distribution

Interactive filters for year, country, type, and rating

These filters allow users to explore the data dynamically and focus on specific segments of Netflix content.

Key Insights

Some interesting insights discovered during the analysis include:

Movies make up a larger portion of Netflix’s catalog compared to TV shows.

There was a noticeable increase in content production starting around 2016.

The United States contributes the highest number of titles on Netflix.

Drama and international content appear among the most frequently produced genres.
